1、 在 Wakatime 安装自己 IDE 工具，比如 VScode

2、 在自己本地的 VSCode 安装 WakaTime 插件⌘ + Shift + P

利用 vercel 自己部署

1、 在vercel用 github 登陆

2、 fork github-readme-stats

3、 点击New Project选择你仓库下的github-readme-stats

4、 配置环境变量PAT_1,这一步很重要，在部署前先配置（我之前都是在构建前再去配置，导致没有生效，这里谁有解答可以告知）

NAME 是PAT_1,Value是github的个人授权token

5、创建Personal access token。

在 github ->settings->Developer settings>Personal access tokens->New personal access token，最后点击Generate Token即可

6、 点击查看 domain，可以自定义
